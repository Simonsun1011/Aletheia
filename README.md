# Aletheia

面向中期美股科技投资的**研究与判断支持系统**。

收束市场信息、呈现量化状态、组织正反证据、记录个人判断、事后对照结果。  
**不是**预测器、信号机或自动交易工具——判断权永远在人。

AI 只做摘要、结构化和知识讲解，**禁止**输出买卖建议、多空立场、目标价或仓位建议。

---

## 功能（当前可用）

| 模块 | 做什么 |
|------|--------|
| **信息流** | 盘后批量卡片：发生了什么（事实摘要），术语可悬停查阅 |
| **操作台** | 按持仓/标的查看宏观 · 基本面 · 技术面 · 叙事扫描（多方 / 中立事实 / 空方） |
| **标的快照** | 经典公开指标（ATR、RSI、布林等），不打综合评分 |
| **术语库** | 中英别名、正文自动高亮、教科书级解读 |
| **判断与复盘** | 判断记录 append-only（可修正/撤回，不可覆盖删除）；复盘与校准页 |
| **设置** | 术语学习状态、Obsidian 导出目录等 |

个人判断 / 随感 / 成交 / 买入方案**只存本机**，不进入本仓库。

## 用法（本地运行）

**环境**：Python 3.10+、Node.js 18+、macOS / Linux 均可。

```bash
git clone https://github.com/Simonsun1011/Aletheia.git
cd Aletheia

# 后端
python3 -m venv .venv
.venv/bin/pip install -r backend/requirements.txt
cp .env.example .env
# 编辑 .env：至少配置 AI 相关密钥（OPENAI_API_KEY 等），按你自己的中转/供应商填写

# 前端
cd frontend && npm install && cd ..

# 一键启动（后端 127.0.0.1:8000，前端 127.0.0.1:3000）
./scripts/dev.sh
# 端口被占用时：./scripts/dev.sh --kill-ports
```

浏览器打开：http://127.0.0.1:3000

请使用**你自己的** API Key，不要共用他人密钥。
